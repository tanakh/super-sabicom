# Super Sabicom

A portable SNES / Super Famicom emulation library written in Rust.
Almost full SNES hardware features are implemented.
Currently, no enhancement chips are implemented.

## Unimplemented features

* PPU
  * [ ] BG Mode 6
  * [ ] Direct Color Mode
  * [ ] Interlaced Mode

* Enhancement chips
  * [ ] Super FX
  * [ ] CX4
  * [ ] DSP
  * [ ] S-DD1
  * [ ] S-RTC
  * [ ] SA1
